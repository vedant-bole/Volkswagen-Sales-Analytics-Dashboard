# Volkswagen India Sales Analytics Dashboard

## Project Overview
This project is an end-to-end data analytics solution developed using Python (Pandas, Matplotlib) and Power BI to analyze Volkswagen India sales data. The objective is to convert raw sales data into meaningful business insights that support data-driven decision-making.

The project focuses on analyzing sales performance, customer behavior, vehicle model trends, revenue distribution, and customer satisfaction metrics through structured data analysis and interactive dashboards.

---

## Tools and Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Power BI
- DAX (Data Analysis Expressions)

---

## Project Workflow
1. Data collection from CSV dataset
2. Data cleaning and preprocessing using Python
3. Exploratory data analysis (EDA)
4. Feature engineering for better insights
5. Data visualization using Matplotlib
6. Dashboard development using Power BI

---

## Key Insights
- Analysis of total sales and revenue performance across regions
- Identification of best-performing Volkswagen models
- Customer segmentation based on age, gender, and occupation
- Evaluation of sales channels such as online, showroom, and referrals
- Analysis of customer satisfaction scores
- Study of repeat customer behavior

---

## Power BI Dashboard Features
- KPI metrics including total sales, revenue, average price, and customer satisfaction
- Sales trend analysis over time
- Model-wise and segment-wise performance analysis
- State and city-level sales distribution
- Interactive filters and slicers for dynamic analysis

---

This project demonstrates practical skills in data analysis, visualization, and business intelligence. It highlights the ability to work with real-world datasets and derive actionable insights for business use cases in the automotive industry.
